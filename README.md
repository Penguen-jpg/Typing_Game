# Typing_Game
使用組合語言所做的打字遊戲

## 注意事項
1. txt裡的每個單字(包括最後一個)都要在結尾加上空格
2. 輸入txt的內容時，不用換行，否則讀取時可能會出錯
3. 測試時不要將資料夾放在系統槽，否則可能會出現權限不足無法開檔的問題
4. 如果要修改題目，單字數量需保留在25個
